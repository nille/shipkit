# Skill Loading Rules

## Critical Rule

**ALWAYS read the relevant SKILL.md file FIRST before taking any action.**

When a user request matches a skill (based on the skill descriptions in context):
1. Read the SKILL.md file completely
2. Follow the workflow defined in the skill
3. Only then proceed with execution

Do NOT:
- Execute a skill without reading it first
- Explore the filesystem to "figure out" how things work
- Make assumptions about file formats or locations
- Start executing before understanding the skill's workflow

If in doubt whether a skill applies, read it first anyway. It's better to read a skill and not need it than to miss important instructions.
